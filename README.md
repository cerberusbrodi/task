# Небольшие задачи на Python

Простой репозиторий для хранения маленьких упражнений и скриптов на Python.  
Создан для практики и быстрых проверок идей — ничего серьёзного, только полезные мелочи.

## Содержание
- `tasks/` — папка с отдельными задачами (каждая задача в своём файле или в своей папке)  
- `examples/` — короткие примеры кода и сниппеты  
- `utils/` — вспомогательные функции, если понадобятся  
- `README.md` — этот файл
